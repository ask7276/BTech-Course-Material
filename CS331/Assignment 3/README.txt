Commands to run code:

Question 1: 

$ ghc -o Q1 Q1.hs
$ ./Q1

This will prompt to enter the value whose square root has to be found. After entering the value, it will display the square root.


Question 2: 

$ ghc -o Q2 Q2.hs
$ ./Q2

This will prompt to enter the index. After entering the value, it will display the index th fibonacci number.


Question 3: 

$ ghc -o Q3 Q3.hs
$ ./Q3 space_seperated_list_of_numbers

It will display the sorted list.

For example: 
$ ghc -o Q3 Q3.hs
$ ./Q3 5 6 4 9 7
